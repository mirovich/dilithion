How to Start Mining

Getting started takes under 60 seconds:

  1. Download the Dilithion software for your platform (Windows, Linux, or macOS)
  2. Extract the archive to a folder
  3. Run START-MINING.bat (Windows) or ./start-mining.sh (Linux/Mac)
  4. Done! Your node syncs, creates a wallet, and starts mining automatically

No configuration needed. No external dependencies. The software auto-detects your CPU cores and connects to the seed node network automatically.

Wallet Features

  • HD Wallet: BIP39 seed phrases (12-word mnemonic backup)
  • BIP44 Key Derivation: Generate unlimited addresses from one seed
  • Encrypted Storage: Wallet file is AES-encrypted on disk
  • Web Wallet: Browser-based wallet for easy access
  • CLI Wallet: Command-line wallet for advanced users
  • Quantum-Safe: All keys use CRYSTALS-Dilithium3 signatures
